## Getting Started

1. Clone the repository  
2. Run `npm install`  
3. Run `npm run dev`  

---

## Features

- React + Vite modern setup  
- Tailwind CSS for styling  
- Framer Motion for scroll animations  
- SEO ready with `react-helmet-async`  
- Fully responsive for desktop and mobile
